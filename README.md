# coq-belgames-csl2023

Coq proof for "Games of Incomplete Information: a Framework Based on Belief Functions"

## Dependencies

This library has been tested with:

- `coq` 8.15.0
- `coq-mathcomp-ssreflect` 1.13.0
- `coq-mathcomp-algebra` 1.13.0

## Build the library

After installing Coq and MathComp, you should just need to type:
```bash
$ ./configure && make && make install
```
